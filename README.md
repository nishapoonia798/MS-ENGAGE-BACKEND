# MS-ENGAGE-BACKEND
This is the back-end for front-end of [A.I. based attendance system](https://github.com/nishapoonia798/MS-ENGAGE-FRONTEND). This python based back-end handles authentication, storing data-set of students and compare images sent. It uses OpenCV and [face-recogniton library](https://pypi.org/project/face-recognition/ "face-recogniton library") to handel face related operations. It uses MongoDB as databse where all the relevant data is stored.
